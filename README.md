# StrongMethod

## 一种面向有限病例罕见病核心基因挖掘的生物功能驱动方法

StrongMethod 是一种基于生物功能的核心基因挖掘方法，主要用于样本量有限的罕见病研究。

传统转录组分析方法通常依赖较大的样本队列，而罕见病由于临床样本数量有限，常规分析方法往往难以发挥理想效果。StrongMethod 旨在通过整合单样本差异表达分析、逐步基因集交集、生物功能富集分析和二次验证，解决有限样本条件下核心基因挖掘困难的问题。

StrongMethod 的核心思想是：在罕见病样本中寻找具有稳定疾病相关表达变化的基因，同时通过生物功能注释提高最终基因集的生物学解释性。

## 方法概述

不同于将所有罕见病样本作为一个整体与常见病样本进行比较，StrongMethod 对每一个罕见病样本分别与常见病样本进行差异表达分析。

随后，将每个罕见病样本获得的差异表达基因集进行逐步交集分析，筛选出在多个罕见病样本中稳定出现的共同基因，这些基因被定义为初始核心基因集。

为了进一步提高结果的生物学解释性，StrongMethod 对非核心基因进行功能富集分析，并提取与通路相关的基因，构建具有生物功能信息的非核心基因集。之后，利用预留的罕见病样本进行二次挖掘，最终获得更加精炼的、具有生物学意义的核心基因集。

## 主要特点

- 适用于样本量有限的罕见病研究
- 适用于转录组测序数据分析
- 通过单样本差异表达分析识别稳定核心基因
- 通过逐步交集减少个体差异的影响
- 结合生物功能富集分析提高结果解释性
- 支持使用预留罕见病样本进行二次验证
- 可与其他小样本基因优先级排序方法进行比较

## 方法流程

StrongMethod 主要包括以下几个概念步骤：

1. 将每一个罕见病样本分别与常见病样本进行比较。
2. 获得每个罕见病样本对应的差异表达基因集。
3. 对多个差异表达基因集进行逐步交集，获得初始核心基因集。
4. 对非核心基因进行生物功能富集分析。
5. 提取通路相关基因，构建非核心基因集。
6. 使用预留罕见病样本进行二次挖掘。
7. 获得最终具有生物学意义的核心基因集。

## 应用场景

StrongMethod 可用于罕见病转录组研究、罕见肿瘤亚型分析，以及其他小样本生物医学数据分析任务。

当样本数量有限，导致传统大队列分析方法、机器学习方法或共表达网络分析方法难以应用时，StrongMethod 可以作为一种具有生物学解释性的核心基因挖掘策略。

## 研究示例

在原始研究中，结直肠癌被用作常见病参考，结直肠印戒细胞癌被用作罕见病示例。

该方法进一步通过 RankProd、Robust Rank Aggregation 以及外部黏液腺癌数据集进行了验证。结果表明，StrongMethod 能够在有限样本条件下识别稳定且具有生物学意义的核心基因。
